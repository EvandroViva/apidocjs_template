# Apidocjs Template
Template ApiDocJS

## Installation

### Package

via npm:

```bash
$ npm install git://github.com/EvandroViva/apidocjs_template --save 
```

### To use

Install apidoc and insert this script into your package file

```bash
"scripts": {
  "postinstall": "apidoc -i routes -o docs -t node_modules/apidocs-template/template/"
}
```
